# 说明

## 1. 启动： 
  - 安装node, 本地18.12.1
  - 安装pnpm: npm i pnpm -g
  - 安装依赖: pnpm i
  - 启动happy-h5: pnpm run happy:h5
  - 启动happy-admin: pnpm run happy:admin


## 2. 项目集目录

- packges
  - happy-admin 订婚后台管理
  - happy-h5 订婚h5
  - editor 富文本组件ckeditor5(未来封装)
  - request axios请求(未来封装)
  - theme-color 学习antd颜色主题算法(未来)
